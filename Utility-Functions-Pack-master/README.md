# Utility Functions Pack — сборка DLL

Сборка выполняется исключительно через **Visual Studio** (MSVC). MinGW, GCC и Clang не поддерживаются: проект использует ATL, `__asm` и MSVC-специфичные расширения.

---

## Требования

- Visual Studio 2022 (Community или выше)
- Компоненты (устанавливаются через Visual Studio Installer):
  - **Разработка классических приложений на C++**
  - **C++ ATL для последних средств сборки (x86 и x64)**

---

## Создание проекта

Файл `.sln` в репозитории отсутствует — проект создаётся вручную.

1. **File → New → Project** → шаблон «Библиотека динамической компоновки (DLL)», язык C++.
2. Скопировать все файлы репозитория в папку проекта, сохранив структуру подкаталогов. Файлы `dllmain.cpp` и `pch.h`, созданные Visual Studio автоматически, заменяются файлами из репозитория.
3. В **Обозревателе решений**: правая кнопка на проекте → **Добавить → Существующий элемент** → выделить все `.cpp`-файлы из всех подкаталогов.

## Настройка свойств проекта

Открыть: правая кнопка на проекте → **Свойства**. Убедиться, что выбраны конфигурация **Release** и платформа **Win32 (x86)**.

**Общие:**

| Параметр | Значение |
|---|---|
| Стандарт языка C++ | ISO C++20 (/std:c++20) |
| Набор символов | Использовать многобайтовый набор символов |

**C/C++ → Создание кода:**

| Параметр | Значение |
|---|---|
| Библиотека времени выполнения | Multi-threaded (/MT) |

**C/C++ → Общие:**

| Параметр | Значение |
|---|---|
| Проверки SDL | Нет (/sdl-) |
| Дополнительные каталоги включаемых файлов | `$(ProjectDir);$(ProjectDir)inc;$(ProjectDir)structs;$(ProjectDir)wrapped-classes` |

**Компоновщик → Ввод → Дополнительные зависимости:**
```
Shell32.lib;Ole32.lib;
```

## Исключение файлов из сборки

Следующие файлы включаются через `#include` внутри `dllmain.cpp` и не должны компилироваться как самостоятельные единицы трансляции. Для каждого из них: правая кнопка → **Свойства** → конфигурация «Все конфигурации» → **Общие → Исключён из сборки → Да**.

Файлы для исключения:
- `TList_methods.cpp`
- `src/DateTime.cpp`
- `src/Tests.cpp`
- `wrapped-classes/ClassTools.cpp`

## Сборка

Конфигурация **Release**, платформа **x86** → **Сборка → Собрать решение** (`Ctrl+Shift+B`).

Готовый файл: `Release\CustomFunctions.dll`

## Сборка с поддержкой Windows 7

Единственное необходимое изменение — замена `std::format` (C++20) на `swprintf`, после чего стандарт языка можно понизить до C++17:

**Свойства → Общие → Стандарт языка C++:** ISO C++17 (/std:c++17)

**Замена в исходном коде:**

```cpp
// До:
std::wstring float_str = std::format(L"{:.{}f}", value, decimal_places);

// После:
wchar_t buf[64];
swprintf(buf, 64, L"%.*f", decimal_places, value);
std::wstring float_str = buf;
```
